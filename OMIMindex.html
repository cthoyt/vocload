OMIM Vocabulary Load Data and Logs

OMIM Vocabulary Load (vocload) Data, Runtime Files, Reports, and Log Files

Inputs

  • omim.txt - OMIM input file
  • OMIM.translation - OMIM-to-MGI Term Translation file
  • OMIM.special - OMIM-to-MGI Word Translation file

Reports

  • discrepancy.html - Curator Report
  • OMIMqc1.rpt - Curator Report
  • OMIMqc2.rpt - Curator Report

Outputs

  • fullLog.txt - File generated by vocload containing a complete record of the job, including all SQL executed.
  • OMIM.tab - File generated by vocload containing all the terms being processed.
  • All available run-time files - All files located in the run-time directory.
