Disease Ontology (DO) Vocabulary Load Data and Logs

DO Vocabulary Load (vocload) Data, Runtime Files, Reports, and Log Files

Inputs

  • download directory - original downloaded files
  • Termfile - DO input file; copy from download directory

Reports

  • discrepancy.html - Curator Report

Outputs

  • fullLog.txt - File generated by vocload containing a complete record of the job, including all SQL executed.
  • DO.tab - File generated by vocload containing all the terms being processed.
  • All available run-time files - All files located in the run-time directory.
